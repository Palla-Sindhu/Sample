# User Registration Page

This project is a simple and responsive user registration form built using HTML and CSS. The form includes fields for a user's name, email, and password, along with a submit button.

## Features

- **Responsive Design**: The registration form is centered on the screen and adapts to various screen sizes, making it accessible on both desktop and mobile devices.
- **Input Validation**: The form includes basic validation to ensure all fields are filled out before submission.
- **Modern Styling**: The page is styled with embedded CSS to provide a clean and user-friendly interface.

## Code Structure

- **HTML Structure**: The HTML file contains the main structure of the registration form, including the input fields and the submit button.
- **CSS Styling**: Basic CSS is included within the HTML file to style the form elements, ensuring a consistent look and feel across different browsers.
